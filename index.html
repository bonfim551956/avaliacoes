🎭 Modo de Operação

O sistema opera em Modo Mock enquanto a Google Business Profile API não é aprovada. Isso permite testar todo o fluxo (busca automática, detecção, geração de resposta) com avaliações fictícias.

Quando o Google aprovar o acesso, basta desativar o modo Mock e configurar o ID da localização de cada loja.

🔐 Conexão OAuth com Google

Conecte sua conta Google que é Owner/Manager das lojas no Google Meu Negócio. O sistema vai usar essa conexão pra buscar avaliações e responder automaticamente.

🏪 Mapeamento de Lojas Google

Conecte cada loja física com seu perfil no Google Meu Negócio. Necessário para o modo REAL funcionar.

🔄 Sincronização

Busca avaliações novas no Google e processa automaticamente (detecta consultor + gera resposta para 5★).
